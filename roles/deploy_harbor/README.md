deploy_harbor
=========

Downloads the requested version of Harbor and deploys it to your cluster.

Role Variables
--------------

| Variable             | Has Default | Example                                                      | Description                                   |
| -------------------- | ----------- | ------------------------------------------------------------ | --------------------------------------------- |
| harbor_version | Yes         | 1.5.0 | The version of Harbor to deploy.
